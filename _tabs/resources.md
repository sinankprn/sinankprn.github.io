## Tools & Libraries

**Machine Learning**
- [Scikit-learn](https://scikit-learn.org/) - Essential ML library for Python
- [PyTorch](https://pytorch.org/) - Deep learning framework
- [LangChain](https://langchain.com/) - Framework for building LLM applications

**Data Analysis**
- [Pandas](https://pandas.pydata.org/) - Data manipulation and analysis
- [Polars](https://pola.rs/) - Fast DataFrame library

**Visualisation**
- [Matplotlib](https://matplotlib.org/) - Python plotting
- [Seaborn](https://seaborn.pydata.org/) - Statistical data visualisation
- [Plotly](https://plotly.com/) - Interactive visualisations
